# 📋 [PRD 업데이트] 가죽 아우터 온라인 수선 사전 접수 시스템

## 1. 핵심 요구사항 변경 내역

### 1-1. 거래처용 접수 페이지 (Front-end) : **'조회' 기능 추가**

단일 URL 접속 시, 화면을 [신규 수선 접수]와 [접수 내역 조회] 두 가지 탭으로 분리합니다.

* **[접수 내역 조회] 탭:**
* **본인 확인:** 별도 로그인 없이 접수 시 입력했던 '매장 담당자 연락처'와 **'매장명'**(또는 고객이름)을 입력하여 본인이 접수한 건만 조회할 수 있도록 구성.
* **조회 화면 노출 데이터:**
* 접수일자, 고객이름, 모델명/스타일번호
* **현재 처리 상태:** (예: 사전판정 중 / 판정 완료 / 수선 진행 중)
* **예상 수선비:** (공장에서 입력한 금액 노출, 예: 30,000원)
* **수선비 내역 및 코멘트:** (예: 지퍼 교체 1.5만, 가죽 찢김 복원 1.5만 / 송아지 가죽 원단 수급으로 2주 소요 예상)
* *(선택)* 수선 진행 여부 확정 버튼: 안내된 비용을 확인한 매장 직원이 [이 비용으로 진행 확정(택배 발송)] 또는 [수선 취소] 버튼을 눌러 피드백을 줄 수 있는 기능.


### 1-2. 관리자 페이지 (Back-end) : **비용 입력란 추가**

* **판정 결과 입력 기능 강화:**
* 수선 상태를 `[사전판정 완료]`로 변경 시, 팝업이나 입력창이 활성화됨.
* **수선비 입력칸:** 숫자로 금액을 기입 (무상 수선일 경우 '0'원 기입).
* **수선 내역 상세(코멘트) 입력칸:** 거래처가 비용을 납득할 수 있도록 간단한 사유나 추가 안내 사항(예: 가죽 이색 발생 가능성 등)을 적을 수 있는 텍스트 칸.


## 2. 변경된 사용자 여정 (User Flow)

1. **[거래처]** 스마트폰으로 URL에 접속하여 수선 내용과 사진을 등록 ➔ **[접수 완료]**
2. **[공장]** 관리자 페이지에서 사진과 요청 내용을 확인.
3. **[공장]** 수선 가능 여부를 판단하고, 시스템에 예상 수선비(금액)와 상세 내역(코멘트)을 입력한 뒤 상태를 `[사전판정 완료]`로 변경.
4. *(개별 연락)* 공장에서 거래처에 "수선 판정이 완료되었으니 접속해서 비용을 확인해 주세요"라고 간략히 알림 (문자나 메신저).
5. **[거래처]** URL의 **[접수 내역 조회]** 탭에 연락처를 입력하고 들어가서 **수선비와 코멘트를 확인**.
6. **[거래처 ➔ 공장]** 비용 확인 후 수선 진행이 확정되면, 해당 의류를 공장으로 택배 발송.
7. **[공장]** 실물 수령 후 상태를 `[실물 입고]` 및 `[수선 진행 중]`으로 변경하여 거래처가 실시간 상황을 조회할 수 있게 함.
